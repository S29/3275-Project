HOW TO START THE APPLICATION:
1. Run setup.exe found in the WarehouseManager\Installer.
2. Follow the prompts.
3. The application should now be installed on your machine.

The source files of the application can be found in WarehouseManager\WarehouseManager. Please be aware that the application inside this folder is currently coded to connect to a specific server. Modifications will need to be made to the source code in order for it to run properly.
